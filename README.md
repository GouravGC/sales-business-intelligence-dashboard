This project transforms raw sales data into actionable business intelligence using Python, Pandas, DuckDB, and Plotly, presented through an interactive Streamlit dashboard.

## Project Overview

The primary goal of this project is to provide a comprehensive analytical framework covering key performance indicators (KPIs), trend analysis, geographical and product performance, customer behavior, and the impact of discounts and shipping modes. The insights derived are crucial for strategic decision-making and improving overall business performance.

## Features

- **Interactive KPI Cards:** Quickly view key business metrics such as total revenue, profit, orders, and average order value.
- **Time-Series Analysis:** Explore yearly, monthly, and quarterly revenue and profit trends with interactive Plotly charts.
- **Geographical Analysis:** Understand sales and profit distribution across regions, states, and cities.
- **Product Performance:** Identify top-performing and underperforming products, categories, and sub-categories.
- **Customer & Segment Analysis:** Gain insights into customer loyalty, top customers, and segment performance.
- **Impact Analysis:** Analyze the effects of discounts and shipping modes on sales and profit.
- **Data Visualizations:** All analyses are supported by interactive Plotly charts for better understanding.
- **Data Download:** Ability to download processed datasets and summary reports directly from the application.

## Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    cd your-repo-name
    ```

2.  **Create a virtual environment (recommended):**

    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows: .venv\Scripts\activate
    ```

3.  **Install the dependencies:**

    ```bash
    pip install -r requirements.txt
    ```

4.  **Ensure `artifacts` folder is populated:**

    This Streamlit application relies on an `artifacts` folder generated by a preceding analytics notebook. Please ensure this folder and its contents (processed data, KPI JSONs, summary tables, Plotly HTML files) are present in the root directory of the project.

    Alternatively, you can run the provided Google Colab notebook to generate all necessary artifacts.

## Usage

To run the Streamlit application:

```bash
streamlit run app.py
```

Open your web browser and navigate to the local URL provided by Streamlit (usually `http://localhost:8501`).
